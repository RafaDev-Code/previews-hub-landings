# Base Wireframe

Esta carpeta es la fuente estructural neutral. No es un template visual ni una landing final.

## Qué contiene

- `wireframes.html`: canvas navegable con el wireframe base.
- `wireframe-kit.jsx`: primitives compartidas, datos ficticios y componentes chicos.
- `app.jsx`: una landing base en mobile y desktop.
- `menu-qr.jsx`: pantalla auxiliar de menú QR mobile-first.
- `design-canvas.jsx`: wrapper tipo canvas para ver artboards.

## Estructura fija de la landing

1. Header
2. Hero
3. Info del local
4. Destacados
5. Galería
6. Menú QR
7. Reservas
8. CTA final

La regla para los próximos pasos: los templates pueden cambiar estilo visual, densidad, ritmo y composición fina, pero deberían respetar este orden funcional.

## Lectura de producción

- `base-wireframe`: base estructural única.
